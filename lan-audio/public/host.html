Routing

1. Keep Windows default playback on your real speakers or headphones, such as AudioBox USB 96.

2. In Windows Volume Mixer, route only the source app you want to stream to CABLE Input (VB-Audio Virtual Cable).

3. On this page, select CABLE Output as the source.

4. Select your real speakers or headphones as the monitor output and set the delay to roughly match the phone latency.

5. The phone hears the undelayed feed. Only the local monitor follows the delay slider.
